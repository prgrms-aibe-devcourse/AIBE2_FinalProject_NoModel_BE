[[overview]]
= Overview

NoModel API는 AI 모델 관리 및 검색 서비스를 제공하는 RESTful API입니다.

[[overview-http-verbs]]
== HTTP Verbs

|===
| Verb | Usage

| `GET`
| 리소스 조회

| `POST`
| 새 리소스 생성

| `PUT`
| 기존 리소스 전체 수정

| `PATCH`
| 기존 리소스 부분 수정

| `DELETE`
| 리소스 삭제
|===

[[overview-http-status-codes]]
== HTTP Status Codes

|===
| Status code | Usage

| `200 OK`
| 요청 성공

| `201 Created`
| 새 리소스 생성 성공

| `204 No Content`
| 요청 성공, 응답 본문 없음

| `400 Bad Request`
| 잘못된 요청

| `401 Unauthorized`
| 인증 실패

| `403 Forbidden`
| 권한 없음

| `404 Not Found`
| 리소스를 찾을 수 없음

| `409 Conflict`
| 리소스 충돌

| `500 Internal Server Error`
| 서버 내부 오류
|===
